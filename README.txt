=== This is a starting template for future projects which includes: ===

** FILES
- package.json file
- .gitignore file
- config.env file
- app.js file (contains app middlewares, routes, controllers, global error handler, etc)
- base.pug file (main view file)

** FOLDERS
* Public folder with 3 sub-folders: stylesheets (CSS), JS & uploads (imgs)
* MVC folders
* Routes folder

** FEATURES
- JavaScript files bundler (parcel)
- Global Error Handler

=== What you must create for each project: ===

* A postman collection
* A mongoDB cluster
* A database connection
